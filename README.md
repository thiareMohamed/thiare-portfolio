## 🎯 À propos

Portfolio personnel moderne avec un design futuriste, créé avec React et Tailwind CSS. Ce portfolio présente mes compétences, projets, expériences professionnelles et formations de manière élégante et interactive.

### ✨ Caractéristiques principales

- 🎨 **Design futuriste** avec effets glassmorphism, néon et animations fluides
- 📱 **100% Responsive** - Optimisé pour mobile, tablette et desktop
- ⚡ **Performance optimisée** - Chargement rapide et animations fluides
- 🌙 **Thème sombre moderne** avec gradients animés
- 🎭 **Animations interactives** - Effets hover, transitions et animations au scroll
- 🔍 **Navigation fluide** - Scroll smooth et détection automatique des sections
- 📧 **Formulaire de contact** - Système de contact intégré

## 📦 Installation

### Prérequis

- Node.js (version 14 ou supérieure)
- npm ou yarn

### Étapes d'installation

1. **Cloner le repository**
   ```bash
   git clone https://github.com/thiareMohamed/thiare-portfolio.git
   cd thiare-portfolio
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Démarrer le serveur de développement**
   ```bash
   npm start
   ```

4. **Ouvrir dans le navigateur**
   ```
   http://localhost:3000
   ```

## 🎨 Personnalisation

### Modifier les couleurs

Les couleurs principales sont définies dans `tailwind.config.js` et `src/index.css`. Vous pouvez modifier :

- Couleurs principales (sky-400, purple-400, pink-400)
- Gradients
- Effets néon

### Ajouter des projets

Modifiez le fichier `src/assets/data/projects.js` pour ajouter vos projets :

```javascript
{
  id: 8,
  title: 'Mon Nouveau Projet',
  description: 'Description du projet',
  url: 'https://example.com',
  image: require('../images/project.png'),
  category: 'Web',
  stacks: ['React', 'Node.js']
}
```

### Modifier les informations personnelles

- **Hero** : `src/components/Hero.jsx`
- **À propos** : `src/components/About.jsx`
- **Contact** : `src/components/Contact.jsx`
- **Parcours** : `src/components/Parcours.jsx`

## 📱 Responsive Design

Le portfolio est entièrement responsive avec des breakpoints optimisés :

- **Mobile** : < 640px
- **Tablette** : 640px - 1024px
- **Desktop** : > 1024px

Tous les composants s'adaptent automatiquement à la taille de l'écran.

## 🐛 Résolution de problèmes

### Le serveur ne démarre pas

```bash
# Nettoyer le cache
rm -rf node_modules package-lock.json
npm install
npm start
```

### Erreurs de compilation Tailwind

Vérifiez que `postcss.config.js` et `tailwind.config.js` sont correctement configurés.

### Images non chargées

Assurez-vous que les images sont dans `src/assets/images/` et que les chemins sont corrects.
